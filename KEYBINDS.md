# Keybinds

This is a focused list for hyprod features. For the full list, see `dots/.config/hypr/hyprland/keybinds.conf`.

## Core UI
| Keybind | Action |
|---|---|
| `Super + A` | Toggle left sidebar (developer sidebar) |
| `Super + N` | Toggle right sidebar |
| `Super + Tab` | Workspace overview |
| `Super + V` | Clipboard overview |
| `Super + .` | Emoji overview |
| `Super + Q` | Close window |

## Launch-or-Focus (Apps)
| Keybind | Action |
|---|---|
| `Super + Return` | Terminal |
| `Super + E` | File manager |
| `Super + W` | Browser |
| `Super + B` | Firefox (direct) |
| `Super + O` | Obsidian |
| `Super + C` | Code editor |
| `Super + X` | Text editor |
| `Ctrl + Shift + Escape` | Task manager |
| `Super + I` | Settings (Quickshell/System) |

## Scratchpads
| Keybind | Action |
|---|---|
| `Super + Shift + G` | lazygit |
| `Super + Shift + B` | btop |
| `Super + Shift + D` | lazydocker |
| `Super + Alt + C` | Claude Code CLI |

## Pyprland
| Keybind | Action |
|---|---|
| `Ctrl + Super + E` | Expose windows |
| `Ctrl + Super + Z` | Magnify |
| `Ctrl + Super + W` | Window picker |
| `Ctrl + Super + Shift + Space` | Center layout toggle |

## Utilities
| Keybind | Action |
|---|---|
| `Super + P` | Password manager (1Password/Bitwarden) |
| `Super + Alt + P` | Pin window |
